# 终端对话演示脚本

## 准备工作

```bash
# 1. 启动所有服务
docker compose up -d

# 2. 启动终端对话（LangGraph 默认开启）
python terminal_chat.py
```

**演示用户**：`u001`（金牌会员，10个订单，520积分）

**终端效果**：每轮对话会实时显示 LangGraph 9 个节点的执行过程：
```
🎯 识别意图 → 🔍 校验参数 → ⚡ 执行业务 → 🛡 安全检查 → ✍️ 生成回复 → 🪞 反思自检 → ✅ 完成
```

---

## 场景一：问候（15秒）

```
  u001 > 你好
```

> 终端显示节点流转，AI 友好回复并介绍服务范围

---

## 场景二：查所有订单（20秒）

```
  u001 > 我的所有订单
```

> AI 逐条列出 u001 的 10 个订单（订单号、商品名、金额、状态）

---

## 场景三：查会员信息（15秒）

```
  u001 > 我的积分多少
```

> AI 回复：金牌会员 520 积分 10 笔订单（语气比普通会员更热情）

---

## 场景四：按商品名查物流（20秒）

```
  u001 > 智能电饭煲到哪了
```

> AI 模糊匹配 → 返回物流轨迹（仓库打包出库 → 到达驿站 → 本人签收）

---

## 场景五：多轮追问 + 审批 ⭐ 重点（45秒）

```
  u001 > 我要退货
```

> 终端显示 `💬 追问` 节点 → AI：「请问您要退哪件商品呢？」

```
  u001 > 恒温热水壶
```

> `🛡 安全检查` → `⏸️ 等待审批` → AI：「好的，我已收到您的退货申请～为了保障权益，退货需要人工审核确认。已为您提交审核，请稍候。」

> ✅ 同时展示：槽位填充 + 多轮追问 + Human-in-the-Loop 审批

---

## 场景六：政策咨询 / RAG（20秒）

```
  u001 > 退换货的流程是什么样的
```

> AI 走 pgvector 知识库检索 → 返回退换货政策

---

## 场景七：转人工（15秒）

```
  u001 > 转人工
```

> AI 自动创建工单并回复工单编号

---

## 场景八：结束对话（10秒）

```
  u001 > 谢谢，再见
```

> AI 友好道别

---

## 总时长

约 **2分30秒 ~ 3分钟**

| 场景 | 展示的技术点 |

| **场景五** | **槽位填充 + 多轮追问 + 人工审批（LangGraph 核心）** |

| 场景七 | 转人工闭环 + 自动创建工单 |
| 场景八 | 礼貌道别 |
